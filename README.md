#### 1. ReactJS

#### 2. NodeJs/ExpressJs

#### 3. Typescript

#### 4. Redux Toolkit

#### 5. MongoDB and Mongoose

#### 6. Styled Components

#### 7. React Icons

#### 8. Ant Design
